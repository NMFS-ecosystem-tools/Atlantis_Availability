Detailed Description

The Setup Tab 4 allows the user to run the Availability Calculator.

This page contains the GUI definitions for the Availability Calculator Run GUI. From this page the user may run the calculator and either export the calculated values into an existing Atlantis .prm file or to a .csv file.
