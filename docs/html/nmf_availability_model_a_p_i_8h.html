Detailed Description

Contains all of the business (i.e., non-GUI) logic for the Availability value calculations.

This file contains the logic to calculate Atlantis food availability values from the required input data. A matrix is passed by reference to the calculate method and is likewise returned, updated with the newly calculated values.
