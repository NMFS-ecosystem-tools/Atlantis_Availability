Detailed Description

The Setup Tab 3 allows the user to enter and modify Species data.

This tab allows the user to define Species and Diet Composition data. It consists of 2 sub-tabs. The 1st sub-tab allows the user to select the species from an Atlantis Species Codes file that are to be used in the Availability Calculator. After selecting the desired species, the user then enters species-specific meta-data. After the species meta-data have been saved, the user may then go to the Diet Composition sub-tab. This sub-tab allows the user to enter in diet composition values for each predator previously defined in the Species sub-tab. Each row in the Diet Composition table must sum to 1.0.
